# Spacifier

Insert a space between a Chinese character and a western character.

## Installation

```
$ gem install spacify
```

## Usage

Put inputs with pipe to `spacifier`:

```
$ echo "何为跨域？何为JSONP？JSONP技术能实现什么？是否有必要使用JSONP技术？" | spacifier
```

Output:

```
$ 何为跨域？何为 JSONP？JSONP 技术能实现什么？是否有必要使用 JSONP 技术？
```
